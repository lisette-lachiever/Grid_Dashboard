# CSS Grid Dashboard

A **responsive dashboard** built using the **CSS Grid layout system**.

## Features
- Responsive **2-column grid layout** (switches to 1-column on mobile)  
- **Stats panel** spans full width  
- **Calendar panel** spans two rows  
- **Color-coded panels** for different dashboard sections  
- Includes **charts**, **messages**, and **activity lists**

---

## Technologies Used
- **HTML5**  
- **CSS3 (CSS Grid)**  
- **Responsive Web Design**

---

## Setup
1. Download all project files to the same folder.  
2. Add your chart image as **`chart.jpg`**.  
3. Open **`index.html`** in your browser.

---

## CSS Grid Properties Used
- `display: grid`  
- `grid-template-columns`  
- `grid-column` and `grid-row` spanning  
- `gap` for spacing  
- Media queries for responsive design
